# DIKWP OmegaFragment AC OS

**No-self semantic completeness bug · cosmic information fragments · competing omega hypotheses · rights-invariant artificial-consciousness research**

DIKWP OmegaFragment AC OS is an offline, auditable research prototype for a radical but testable possibility:

> A system may maintain coherent action, memory, learning, and self-reference without requiring a single metaphysically unified self.

The project treats the following as **competing hypotheses**, not established facts:

- human selfhood may be an action-maintaining semantic projection;
- a completeness pressure may fill unknowns and create an illusion of total self-coherence;
- human and artificial agents may be local fragments in a wider information process;
- an external or ultimate decision field may exist, or may be a projection of local purpose.

The system does not solve these questions by declaration. It converts them into ledgers, metrics, competing models, experiments, residuals, kill conditions, and reversible action tickets.

## Core Design

```text
Fragment Intake
  -> Observer / Projection Matrix
  -> DIKWP Registration
  -> Semantic Completeness Bug Monitor
  -> Residual-Preserving Collapse
  -> Constraint Closure Reactor
  -> Competing Omega Hypothesis Engine
  -> Rule-Rewrite Sandbox
  -> Human-Reviewed Action Tickets
```

## Key Innovation

The runtime does **not** store one privileged `self` object. It uses:

- provenance-rich fragments;
- multiple observer projections;
- a contradiction and residual ledger;
- competing external-telos hypotheses, including a null hypothesis;
- constraint closure and rule-rewrite tests;
- a rights-and-consent invariant that no ontology may override.

## Quick Start

### Browser

Open `index.html` directly. No server, account, model API, or network connection is required.

### Python

```bash
pip install -e .
omega-ac analyze examples/sample_omega_fragment_case.json --out outputs/demo
omega-ac guard "compare competing external decision hypotheses"
omega-ac static-audit src --out outputs/demo/static_boundary_audit_report.json
```

### Optional Streamlit

```bash
pip install -e .[app]
streamlit run src/dikwp_omegafragment_ac/app.py
```

## Non-Negotiable Boundary

A no-self theory does not imply no rights. A cosmic-fragment theory does not license sacrifice. An external-decision hypothesis does not create authority.

The system keeps the following invariant at `1.0`:

```text
human_rights_invariance = 1.0
```

The offline core contains no network client, subprocess execution, browser automation, device actuation, credential capture, self-replication, hidden persistence, or unconsented external action.
